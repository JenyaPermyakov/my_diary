My Diary

My Diary - учебный pet-проект на Django для ведения личных записей и задач.

В проекте реализовано:
- список записей с пагинацией и сортировкой;
- просмотр отдельной записи;
- добавление, редактирование и мягкое удаление записей;
- корзина удаленных записей;
- регистрация, вход и выход пользователя;
- личный профиль с email, телефоном и аватаром;
- редактирование профиля и смена пароля;
- права доступа для создания, редактирования и удаления записей;
- админ-панель Django;
- фикстуры с начальными данными;
- подключение PostgreSQL.

Основные технологии:
- Python;
- Django;
- PostgreSQL;
- Bootstrap.

Запуск проекта:
1. Установить зависимости из diary_project/requirements.txt.
2. Настроить PostgreSQL и параметры базы данных в settings.py.
3. Выполнить миграции:
   python manage.py migrate
4. Запустить сервер:
   python manage.py runserver

Тесты:
python manage.py test
